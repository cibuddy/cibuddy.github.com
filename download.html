Main Download Locations :: continously updated

Binaries from various versions could be obtained from two different locations basically. Each of these locations will contain the very same artifacts, because they are distributed automatically. Depending on your preferred way of consuming the software you could choose either one.

Github:Download @github.com

Alongside the sources all released versions could be found on the downloads page of the CIBuddy project on GitHub here.

Maven:Download @sonatype

In case you prefer to download CIBuddy from Maven you basically have to choices. Either you go for the official Sonatype repository, which will contain snapshot versions along with released versions or you skip adding the repository and obtain your copy directly from Maven Central (however, this might take a while till a new version is pushed, because of the release approval process of mvn central). 

<repositories>
  <repository>
    <id>sonatypeOSSRepo</id>
    <name>Sonatype OSS Repository</name>
    <releases>
      <enabled>true</enabled>
      <updatePolicy>never</updatePolicy>
      <checksumPolicy>warn</checksumPolicy>
    </releases>
    <snapshots>
      <enabled>true</enabled>
      <updatePolicy>daily</updatePolicy>
      <checksumPolicy>fail</checksumPolicy>
    </snapshots>
    <url>https://oss.sonatype.org/content/groups/public/</url>
    <layout>default</layout>
  </repository>
</repositories>
