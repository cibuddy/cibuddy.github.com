CIBuddy SNAPSHOT :: working copy

The most recent feature and bugfix list could be found here.

The latest version could be obtained from the maven directory here. Just pick the latest SNAPSHOT version available. In case you prefer provisioning an already existing Karaf container, you could of course utilize the CIBuddy feature descriptor found in this location: https://oss.sonatype.org/content/groups/public/com/cibuddy/karaf.features/.
The XML is annotated with comments, making it simple to choose the correct feature to use.

CIBuddy Source Code :: github repo

In case you want more and would like to obtain the source code using GIT, you could simply execute this command (make sure you're on the master branch):

$ git clone git://github.com/cibuddy/cibuddy

Maven Repo :: continously updated

Binaries from various versions could be obtained from two different Maven locations. SNAPSHOT and release are versions are available from the official Sonatype repository, while the released versions are also pushed to Maven Central with a short delay of less than a day.

Maven Sonatype Repository Configuration: 

<repositories>
  <repository>
    <id>sonatypeOSSRepo</id>
    <name>Sonatype OSS Repository</name>
    <releases>
      <enabled>true</enabled>
      <updatePolicy>never</updatePolicy>
      <checksumPolicy>warn</checksumPolicy>
    </releases>
    <snapshots>
      <enabled>true</enabled>
      <updatePolicy>daily</updatePolicy>
      <checksumPolicy>fail</checksumPolicy>
    </snapshots>
    <url>https://oss.sonatype.org/content/groups/public/</url>
    <layout>default</layout>
  </repository>
</repositories>
